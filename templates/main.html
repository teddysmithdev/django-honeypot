Django honeypot is a low-interaction TCP honeypot that logs connections from SSH, Telnet, and ICMP.

Low-interaction Python Honeypot With Web Interface

Seperate honeypot scripts are run independently of Django using Celery, Redis, and Supervisor. Once connection is established network socket allows 1024 bytes of information to be recieved and closes out.
